Aprendiendo sobre el posicionamiento de elementos

Static – Relative – Absolute – Fixed - Sticky

Static

Es el valor de posicionamiento predeterminado este respeta el flujo HTML y las propiedades top – buttom – left – rigth y z-index no funcionan.

Relative

Esta propiedad permite que pueda posicionar un elemento es decir que podemos mover el elemento sin perder su posicionamiento en el flujo esto funciona como el ejemplo:
cuando estamos haciendo una fila para que nos atiendan, cuando aplicamos esta propiedad podemos por ejemplo movernos por ejemplo al baño y no perderemos la posición en la que nos encontramos

Absolute

En esta propiedad el elemento pierde la posición de referencia o la posición actual del flujo HTML Por lo que toma por referencia un elemento padre relativo y si no lo hoy toma el elemento padre

Fixed

Esta propiedad hacemos que el elemento siempre se mantenga en la pantalla es parecido al inmovilizador de columnas en Excel y esta propiedad comparte algunos comportamientos de la absolute ya que se posiciona con respecto a la posición del antepasado relativo en caso que no tenga se posiciona con respecto a la pantalla.

Sticky

Esta propiedad es también conocida como pegajosa es decir que se pega al elemento que lo contiene este se mueve similar al fixed pero se detiene en la parte final del elemento que lo contiene y el fixed.
